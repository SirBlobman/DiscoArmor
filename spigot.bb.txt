[SIZE=5][B]Supported Spigot Versions:[/B][/SIZE]
1.13.2, 1.14.2, 1.15.2, 1.16.1

[SIZE=5][B][COLOR=#FF0000]Description:[/COLOR][/B][/SIZE]
Disco Armor is a fun plugin that adds leather armor with different colors.

[SIZE=5][B][COLOR=#FF0000]Features:[/COLOR][/B][/SIZE]
[LIST]
[*] Seven different disco armor types
[*] Configurable color changing speed
[*] Steal and drop prevention with configurable message
[*] Automatically save and return previous armor if player is damaged
[*] Configurable messages, lores, and armor display name
[/LIST]

[SIZE=5][B][COLOR=#FF0000]Screenshot:[/COLOR][/B][/SIZE]
[IMG]https://i.imgur.com/Ocw601U.gif[/IMG]

[SIZE=5][B][COLOR=#FF0000]Permissions:[/COLOR][/B][/SIZE]
[LIST]
[*][B]disco-armor.command.disco-armor[/B]: Allows access to the '/disco-armor' command.
[*][B]disco-armor.command.disco-armor.reload[/B]: Allows access to the '/disco-armor reload' command.
[/LIST]

[SIZE=5][B][COLOR=#FF0000]Default Config:[/COLOR][/B][/SIZE]
[SPOILER="config.yml"][CODE=YAML]
# How long (in ticks) should the time be between color changes?
# 1 = once per tick
# 20 = once per second
# 1200 = once per minute
#
# Default: 5
armor-speed: 5

# Should disco armor be disabled if a player takes damage?
disable-on-damage: true

# Should disco armor have a LUCK enchantment and a HIDE_ENCHANTS flag?
# (This is the default option, players can change their own armor by typing '/disco-armor glow')
glowing: false

menu-title: "&nDisco Armor Menu&r"
display-name: "&1D&2I&3S&4C&5O&r Armor"
lore:
  - "&7This is an awesome armor set,"
  - "&7but you can only remove it by"
  - "&7typing &e/disco-armor off&7."

# Setting a message to "" or '' will prevent it from being sent.
# DO NOT remove any message paths from this config.
messages:
  enable-broadcast: "&b&ki&1&lD&2&lI&3&lS&4&lC&5&lO&b&ki&r"
  disable-broadcast: "&4&l:(&r"
  not-player: "&cOnly players can use this command."
  prevent-click: "&cYou are not allowed to remove your disco armor."
  change-type: "&aYou changed your armor type to &c{type}&a."
  command-usage: "&f/disco-armor on/off/menu/glow"
  reload-success: "&aSuccessfully reloaded the config.yml for Disco Armor."
  enable-glow: "&aSuccessfully enabled your armor glow effect."
  disable-glow: "&aSuccessfully disabled your armor glow effect."
[/CODE][/SPOILER]
